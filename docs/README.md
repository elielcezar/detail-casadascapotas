# docs — material de apoio (não entra no build do site)

- `conteudo/` — PDFs originais enviados pelo cliente (portfólios de Película, Limpeza e PPF), fonte dos textos do site.
- `referencias/` — screenshots de layouts usados como referência visual.
- `_v1/` — primeira versão do protótipo HTML.
- `img-fora-de-uso/` — imagens que não são usadas no site atual (guardadas para uso futuro).

O protótipo HTML completo que deu origem ao projeto Next.js está preservado na tag git `prototipo-html`.
